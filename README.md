# Django Shop API RESTful

## Application Overview

This project is a guided REST API exercise built with Django REST Framework and currently under development.
It allows managing a shop through multiple endpoints, starting from basic CRUD operations and progressively adding advanced features and authentication for secure access.

## Features

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- Postman

## Installation

### Clone the Repository

- `git clone https://github.com/fabroyer/django-shop-api.git`
- `cd django-shop-api`

### Create and Activate Virtual Environment

Windows:
```bash
python -m venv env
env\Scripts\activate
```

Mac/Linux:
```bash
python -m venv env
source env/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

### Run the Development Server

```bash
python manage.py runserver
```
